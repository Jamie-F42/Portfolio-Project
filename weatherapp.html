This project ended up being two parts. The first was building it with HTML and Javascript, I learned API integration with axios and Bootstrap. Then later I had to convert it into a React application where I learned how to build components and utilize some tools from React such as States and importing components others have made like the icons.

I got to learn how to use objects and arrays in both JavaScript and React.js. I had to retrieve arrays of information from the API and disperse it into my own UI to display the accurate information I wanted.

I like how powerful React can be-especially building things in separate components, and I'm looking forward to learning more about its capabilities.
